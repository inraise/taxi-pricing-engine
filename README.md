# Taxi Pricing Engine — Упрощённая модель ценообразования такси

Учебный проект, моделирующий базовые принципы динамического ценообразования сервиса такси.

## О проекте
- Расчёт базовой цены (расстояние + время)
- Временные коэффициенты (часы пик, ночь, выходные)
- Погодные условия (дождь, снег, сильный ветер)
- Загруженность дорог (пробки)

## Структура

```
taxi-pricing-engine/
├── cmd/
│   └── main.go           # Точка входа
└── internal/
    ├── base_price.go     # Базовая цена
    ├── time_multiplier.go    # Временной коэффициент
    ├── weather_multiplier.go # Погодный коэффициент
    ├── traffic_multiplier.go # Коэффициент пробок
    └── price_calculator.go   # Агрегация и лимиты
```

## Запуск

```bash
go run cmd/main.go
```

Пример вывода:
```
Ваша цена: 350 руб.
```

## Как это работает

1. **Базовая цена**: 10 ₽ за км + 2 ₽ за минуту
2. **Временной коэффициент**:
   - Ночь (0–5 ч): ×1.5
   - Часы пик (7–10 ч, будни): ×1.3
   - Выходные: ×1.2
   - Остальное время: ×1.0
3. **Погодный коэффициент**:
   - Дождь: +12.5%
   - Сильный дождь: +20%
   - Снег: +15%
   - Ветер >15 м/с: +10%
4. **Коэффициент пробок**: зависит от загруженности дорог
5. **Финальная цена** ограничивается диапазоном 99–20 000 ₽
